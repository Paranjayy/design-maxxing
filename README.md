# design maxxing

**430+ web dev projects, animations, and templates — all browsable in one place.**

Browse live iframes, toggle between desktop/tablet/mobile views, search by category, and explore everything without downloading a single file.

## Tech Stack

- **Next.js 15** (App Router, static generation)
- **React 19**
- **Tailwind CSS v4**
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Keyboard Shortcuts (Viewer)

| Key | Action |
|-----|--------|
| `1` | Desktop view |
| `2` | Tablet view (768px) |
| `3` | Mobile view (375px) |
| `F` | Full screen |
| `Ctrl+R` | Refresh |

## Deployment

This app is fully static (SSG). Deploy anywhere:

```bash
npm run build
# Deploy the .next/output folder
```

Or push to GitHub and connect to Vercel for automatic deployments.
