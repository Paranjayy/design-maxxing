## Getting Started

```bash
git clone https://github.com/Paranjayy/design-maxxing.git
cd design-maxxing
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Keyboard Shortcuts (Viewer)

| Key | Action |
|-----|--------|
| `1` | Desktop view |
| `2` | Tablet view (768px) |
| `3` | Mobile view (375px) |
| `F` | Full screen |
| `⌘R` | Refresh |
| `Esc` | Go back |

## Deployment

Fully static (SSG) — deploy anywhere:

```bash
npm run build
```

Push to GitHub and connect to Vercel for automatic deployments.
